翻卡组件 FlipCard

正面问题、背面答案，点击或键盘翻转，支持多张卡片轮播。适合单词记忆、知识点回顾等场景。

⌨️ 键盘方向键切换
🔄 3D 翻转动画
📊 进度指示
🔔 翻转回调
// 引入 JS 和 CSS
new EduInteract.FlipCard({
  el: '#flip-card',
  data: [
    { front: '问题1', back: '答案1' },
    { front: '问题2', back: '答案2' }
  ],
  onFlip: function(index, isFlipped, card) {
    console.log('卡片翻转', index, isFlipped);
  },
  onComplete: function(total) {
    console.log('全部翻完了', total);
  }
});

拖拽配对 DragMatch

左右两列元素，拖拽或键盘操作进行配对。适合词汇匹配、概念关联等互动练习。

🖱️ 拖拽操作
⌨️ 键盘配对
✅ 正误反馈
🔔 结果回调
new EduInteract.DragMatch({
  el: '#drag-match',
  data: [
    { left: '苹果', right: 'apple' },
    { left: '香蕉', right: 'banana' }
  ],
  allowRetry: true,
  onMatch: function(item, isCorrect) {
    console.log('配对结果', isCorrect);
  },
  onComplete: function(correct, total) {
    console.log('完成', correct, '/', total);
  }
});

随堂投票 QuizVote

单选或多选投票，实时显示结果统计。适合课堂测验、互动提问、意见征集等场景。

1️⃣ 单选/多选
⌨️ 数字键快选
📊 结果统计
🔔 提交回调
new EduInteract.QuizVote({
  el: '#quiz-vote',
  data: {
    question: '这道题的答案是？',
    options: ['选项A', '选项B', '选项C'],
    type: 'single',
    correctAnswer: [0]
  },
  showResult: true,
  onVote: function(selected) {
    console.log('选择了', selected);
  },
  onSubmit: function(selected, isCorrect) {
    console.log('提交', isCorrect);
  }
});

进度条 ProgressBar

展示课堂进度、答题进度，支持分段节点标记。可配合其他组件使用。

📈 平滑动画
🏷️ 分段节点
🎨 多种状态色
🔔 变化回调
const progress = new EduInteract.ProgressBar({
  el: '#progress-bar',
  data: {
    value: 0,
    max: 100,
    label: '课堂进度',
    showPercent: true,
    segments: [
      { label: '导入', value: 20 },
      { label: '讲解', value: 60 },
      { label: '练习', value: 90 }
    ]
  },
  onChange: function(value, percent) {
    console.log('进度变化', percent + '%');
  }
});

progress.setValue(50);
progress.animateTo(100, 1000);
